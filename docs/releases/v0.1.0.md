# Release Notes - v0.1.0

发布日期：2026-04-16

## 新增能力
- 可持续的人生分支推演，不是一次性问答。
- 支持 `回退一步` 与 `回退到第N步`，并重算后续路径。
- 探索模式/现实模式切换。
- 轨迹卡统一输出：当前选择、3年后状态、代价、补救动作。

## 内容范围
- 首版覆盖 7 大场景：升学、读研/考公、就业/转行、城市、关系、家庭、创业。
- 内置 14 个节点与关键分支。

## 安全机制
- 高危表达立即中断剧情并进入危机支持模板。
- 心理健康低阈值时强制插入稳定动作建议。

## 安装与验证
- macOS：`bash scripts/install.sh`
- Windows：`powershell -ExecutionPolicy Bypass -File scripts/install.ps1`
- 自检：`python -m engine verify`

## 已知限制
- 当前版本为规则驱动，不接外部统计数据库。
- 仅提供安装+自检，不含自动首轮剧情。
